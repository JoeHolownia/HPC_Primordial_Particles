# HPC Primordial Particles

This is the repository for my 3rd Year High Performance Computing course project.

This project implements the Primordial Particle system described in the article published in Nature: "How life emerges from a simplistic particle motion law" in C++, and performs various parallelisations on it.

Article: https://www.nature.com/articles/srep37969
YouTube Video: https://www.youtube.com/watch?v=makaJpLvbow&ab_channel=ArtificialLifeLabGraz
